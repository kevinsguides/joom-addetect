## Adblock Detector
A free adblock detection plugin for Joomla 4.

Detects some, but not all, ad blockers.

### Known Working With...
As of 6/4/2023 I've verified this is working on...

* AdBlocker Ultimate
* AdBlock
* AdBlock Pro - Mobile Safari

### Known Broken With...
It won't be able to detect...
* UBlock Origin
